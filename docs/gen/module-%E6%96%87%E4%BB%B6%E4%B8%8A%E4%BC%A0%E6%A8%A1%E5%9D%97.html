<inner> uploadFile64(req, res)

自动根据类型识别存放文件夹

Parameters:
Name Type Description
req json

请求参数： file64：图片，语音，视频64位后的编码 type:上传文件类型（0:图片；1:语音；2:小视频）， url:/filedirectupload/uploadFile64
res json

正确返回 { fileType:'commentImg', filename:'images/'+imgfileName+".jpg" }，{ fileType:'voice', filename:'voices/'+voicefileName+".wav" }，{ fileType:'video', filename:'videos/'+videofileName+".mp4" }，错误返回{ err:'上传参数不对', filename:'null' } 或者文件保存错误会返回 错误详情
